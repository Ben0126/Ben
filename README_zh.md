- 💼 在 **工業技術研究院 (ITRI)** 光電系統研究所擔任過工程師實習生
- 🚁 專長領域包括**無人機控制**、**強化學習**與**智慧系統整合**

- 🌐 致力於開發能夠解決實際問題的智能系統
